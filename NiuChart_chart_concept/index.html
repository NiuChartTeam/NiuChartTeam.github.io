了解NiuChart

为了理解NiuChart是如何工作的，有必要先理解图表的各个部分或概念。
下面是这张图片，描述了图表中的主要概念：

canvas

表示绘制图表的画布。
更多细节，请查看canvas

series

表示图表上呈现的一个或多个系列的数据。
更多细节，请查看series

touchTools

当手指触摸在图表的一个系列或一个点上时，你可以得到一个工具提示。工具提示描述了在图表中特定部分的数据。
更多细节，请查看touchTools

axis

图表的x和y轴，也能够为不同的数据系列使用多样的坐标轴。大多数的图表类型，像典型的线性图和柱形图都有坐标轴。甚至有些图表需要上下左右四个坐标轴。饼图、地图、环图、单值图没有坐标轴。
更多细节，请查看axis

selection

刚进入图表时的默认选中值。相关属性如下： 

"selection": {
    "selectionSource": "Y",
    "selectionValue": 0,
    "selectionType": "NXChartSelectionLast"
}

“selectionSource”：选中值的值来源，是Y,还是X,还是Y2
“selectionValue”：选中的值，可以是真实的值或者索引值
“selectionType”：选中的模式。可选项有：”NXChartSelectionFirst”(选中第一根),”NXChartSelectionLast”(选中最后一根),”NXChartSelectionBiggestValue”(选中最大值),”NXChartSelectionSmallestValue”(选中最小值),”NXChartSelectionEqualGreaterThan”(选中大于等于给定值),”NXChartSelectionEqualLessThan”(选中小于等于给定值),”NXChartSelectionSpecificIndex”(选中指定索引值)。

  • 当”selectionType”为”NXChartSelectionSpecificIndex”时，需要给”selectionValue”设置选中值的索引
  • 当”selectionType”为”NXChartSelectionEqualGreaterThan”或”NXChartSelectionEqualLessThan”时，需要给”selectionValue”设置临界值。
  • 当”selectionType”为其他类型时，”selectionValue”的值不用设置，自动设置为相应类型的值。
