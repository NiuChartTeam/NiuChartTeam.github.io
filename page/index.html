report

报表的相关属性如下： 

{
    "id": "",
    "width": 0,
    "height": 0,
    "title": "",
    "subTitle": "",
    "description": "",
    "note": "",
    "creator": "",
    "createDate": "",
    "lastUpdate": "",
    "version": "",
    "metadataStructures": [{
        ...
    }],
    "pages": [{
        ...
    }]
}

“id”：report在列表中的唯一标识符
“width”：指定report的宽度
“height”：指定report的高度
“title”：report的标题
“subTitle”：report的子标题
“description”：对该report的描述
“note”：report的注解
“creator”：report的创建者
“createDate”：report的创建时间
“lastUpdate”：report的更新时间
“version”：report的版本号
“metadataStructures”：report拥有的数据库的数据模型。详情见structure
“pages”：report内包含的所有page的属性。见page

page

瀑布流页的相关属性如下： 

{
    "id": "",
    "index": 0,
    "note": "",
    "pageTitle": "",
    "pageColor": "",
    "cards": [{
        ...
    }]
}

“id”：page在report中的唯一标识符
“index”：page在report中的索引
“note”：page的注解
“pageTitle”：page的标题
“pageColor”：page页的主题色。
“cards”：page页内包含的所有card的属性。见card

card类型不为NXCubeCard时chart的样式
