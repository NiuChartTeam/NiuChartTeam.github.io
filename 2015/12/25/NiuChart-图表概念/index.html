NiuChart 图表概念

发表于 2015-12-25   |  

了解NiuChart

为了理解NiuChart是如何工作的，有必要先理解图表的各个部分或概念。
下面是这张图片，描述了图表中的主要概念：

canvas

表示绘制图表的画布。
更多细节，请查看canvas

series

表示图表上呈现的一个或多个系列的数据。
更多细节，请查看series

touchTools

当手指触摸在图表的一个系列或一个点上时，你可以得到一个工具提示。工具提示描述了在图表中特定部分的数据。
更多细节，请查看touchTools

axis

图表的x和y轴，也能够为不同的数据系列使用多样的坐标轴。大多数的图表类型，像典型的线性图和柱形图都有坐标轴。甚至有些图表需要上下左右四个坐标轴。饼图和地图没有坐标轴。
更多细节，请查看axis

lineStyle

线的样式 

{
    "lineWidth": 1,
    "lineColor": "#000000",
    "lineAlpha": 1,
    "lineCap": "BUTT",
    "lineJoin": "MITER",
    "dashStyle": {
        "intervals": [0, 0, 0, 0],
        "phase": 0
    }
}

fillColor

填充色 

{
    "fillAlpha": 1,
    "fillColor": "#000000"
}

paddingStyle

内边距样式 

{
    "left": 0,
    "top": 0,
    "right": 0,
    "bottom": 0
}

canvas

canvas用来绘制chart的各个部分。以下是canvas元素的简单概述：

canvas属性节点可以为空，使用默认样式。见chartOptions.canvasOptions

graph

图表所在的区域。可以在json配置文件中配置相关的属性节点。
graph相关属性节点可以为空，使用默认样式。见canvasOptions

  • graphBackgroundFill
    graph所在区域的背景填充
    详情见fillColor

  • graphBorderStyle
    graph所在区域的边界线框样式
    详情见lineStyle

  • graphPadding
    graph的内边距
    详情见paddingStyle

plot

series data绘制的区域
plot相关属性节点可以为空，使用默认样式。见canvasOptions

  • plotBackgroundFill
    plot的背景填充
    详情见fillColor

  • plotBorderStyle
    plot边界线框的样式
    详情见lineStyle

chartPadding

图表的内边距
相关熟悉可以为空，使用默认样式。见canvasOptions.chartPadding
详情见paddingStyle

touchTools
